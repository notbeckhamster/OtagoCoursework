## Usage
Within etude_11 directory 

**Compile class files**
```
    javac *.java

```

**Run PegGen Program**
Input: The program expects a positive integer as input, denoted by n
```
    java PegGen n
```

**Run Peg GUI Program**
```
    java SolitaireGUI 
```
